# pm_notif (Уведомление ЛС)

Данный плагин информирует пользователя о новых личных сообщениях...

Возможности плагина:
- интервал показа уведомления
- обрезка текста сообщения
- размера аватара
-------------------
Требования плагина:
- должен установлен плагин pm
